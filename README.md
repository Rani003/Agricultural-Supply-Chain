How to Run

Buka 2 terminal
# Terminal 1 (Backend)
```html
cd agricultural-supply-chain/backend
npm init -y
npm install express cors body-parser uuid
npm start
```
# Terminal 2 (Frontend)
```html
cd ../frontend
npx create-react-app .
npm install axios
npm start
```
# Akses Backend
```html
HTTP://localhost:5000
```
# Akses Frontend
```html
HTTP://localhost:3000
```
